PROJECT TITLE: Cloud Server Application
PURPOSE OF PROJECT: To facilitate the transfer of files between computers within the house.
VERSION or DATE: 1.4.7/August 3 2018
HOW TO START THIS PROJECT: java ./StrmServer
                            OR
                           StrmServer.sh start
AUTHORS: William Ritchie
USER INSTRUCTIONS:
  The StrmClientBE code, and the StrmServer code is set to port# 42843. If you want to use a different port, 
  then you must change the port# in the StrmClientBE.java code, and the port# in the StrmServer.java code to match. If you are using the shell script to run the 
  server as a service, you must also manipulate the variables at the top of StrmServer.sh to your liking.

StrmServer v.1.4.7 or later, compatible with StrmClient v.1.9.2 or later.

CloudServer v1.1.5 to v.1.3.7, compatible with CloudClient v1.6.1 to v.1.8.2.

CloudServer v.1.0.0 to v.1.0.5, compatible with Cloudclient v.1.0.0 to v.1.5.1.
